Tuzet Soil Moisture (Water Potential) Stress

This page documents the Tuzet stress function used to scale stomatal (and/or photosynthetic) capacity as a function of plant water potential. It provides a smooth, sigmoidal decline in conductance with decreasing (more negative) leaf/xylem water potential.

Summary

  • Purpose: compute a water‑status stress multiplier S_ψ ∈ [0,1] from leaf (or xylem) water potential.
  • Inputs: leaf/xylem water potential ψ (MPa), parameters controlling the midpoint and slope.
  • Outputs: scalar stress factor S_ψ applied within stomatal/photosynthesis sub‑models.

Model formulation

Following Tuzet et al. (2003), the water‑potential response is

\[S_\psi(\psi) = \frac{1}{1 + \exp\big[a\,(\psi - \psi_{50})\big]},\]

where:

  • ψ — leaf or xylem water potential (MPa; negative in tension),
  • ψ_{50} — midpoint potential where S_ψ = 0.5,
  • a — dimensionless slope (steepness). Larger a yields a sharper transition.

Some implementations include a residual floor S_{min} so that S_ψ = S_{min} + (1-S_{min})/(1 + exp[a(ψ-ψ_{50})]).

Choice of ψ: Depending on configuration, ψ may be the soil water potential (root‑weighted), stem/xylem, or leaf potential derived from a hydraulic model.

Variables & units

QuantitySymbolUnitsNotes
Water potentialψMPatypically ≤ 0
Midpoint potentialψ_{50}MPawhere S_ψ = 0.5
Slopea–logistic steepness
Stress multiplierS_ψ–0 (full stress) … 1 (no stress)

Parameters

ParameterDescriptionTypical value/range
ψ50midpoint water potential−2 to −1 MPa (species/site dependent)
aslope parameter2–12
Sminresidual floor0.0–0.1 (optional)
state_variablewhich potential to use (:leaf, :xylem, :soil):leaf (common)

Exact field names/types should match soil_moisture_stress.jl (Tuzet config/struct).

Numerical details & coupling

  • ψ is read from the hydraulics/soil module at each step (or derived diagnostically).
  • S_ψ multiplies gₛ and, optionally, biochemical capacities.
  • The logistic form is smooth/differentiable; no special smoothing is required.

Using the Tuzet stress in a standalone run

using ClimaLand

tuzet = Vegetation.SoilMoistureStress.Tuzet(
    ψ50 = -1.5,   # MPa
    a   = 6.0,    # unitless slope
    Smin = 0.0,   # optional residual
    state_variable = :leaf, # which ψ to use
)

veg = Vegetation.CanopyConfig(
    soil_moisture_stress = tuzet,
    # ... other vegetation settings
)

model = ClimaLand.Standalone.CanopyModel(vegetation = veg)
sol = ClimaLand.solve!(model; tspan = (0.0, 86400.0))

d = ClimaLand.diagnostics(sol)
Sψ = d[:soil_moisture_stress]    # dimensionless multiplier
ψ  = d[:leaf_water_potential]    # MPa (name may vary)

Output diagnostics

  • :soil_moisture_stress — current stress multiplier S_ψ (–)
  • :leaf_water_potential (or :xylem_water_potential) — water potential used (MPa)
  • Parameters are echoed in metadata for traceability (implementation‑dependent).

Tips

  • Calibrate ψ50 and a from pressure‑volume/stomatal response curves for your species.
  • Ensure consistency of sign convention (MPa, negative values for tension).
  • If coupling with a dynamic hydraulic model, verify timestep adequacy around dawn/dry‑down transitions.

References

  • Tuzet, A., Perrier, A., Leuning, R., 2003. A coupled model of stomatal conductance, photosynthesis and transpiration. Plant, Cell & Environment 26, 1097–1116.
  • Leuning, R., 1995. A critical appraisal of a combined stomatal–photosynthesis model. Plant, Cell & Environment.
